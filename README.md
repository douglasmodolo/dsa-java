## 📋 Tópicos

### ☕ Java / Algoritmos

#### 1. Strings
Manipulação e análise de cadeias de caracteres.
- Anagramas
- Palíndromos
- Inversão de strings
- Remoção de duplicatas
- Contagem de caracteres

#### 2. Arrays e Listas
Operações fundamentais sobre coleções de dados.
- Busca linear
- Busca em intervalo
- Filtros e transformações
- Remoção e inserção de elementos

#### 3. HashMap e HashSet
Uso de estruturas baseadas em hash para buscas eficientes.
- Contagem de frequência
- Verificação de existência
- Agrupamento de dados
- Eliminação de duplicatas

#### 4. Ordenação e Busca
Algoritmos clássicos e suas aplicações.
- Busca binária
- Bubble Sort
- Merge Sort
- Comparação de complexidades

#### 5. Lógica Geral
Problemas clássicos de lógica e condicionais.
- FizzBuzz e variações
- Números pares e ímpares
- Sequências e padrões
- Manipulação de números
